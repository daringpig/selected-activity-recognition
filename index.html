This project is about Linear acceleration data from triaxial sensor used for classifying selected human activities. Selected activities, such as running, jogging, walking and idle are classified. The application made for Android, uses K3DH 3-axis acceleration sensor data for classifying the mentioned activities using Machine learning & Datamining algorithms.

Setup instructions

  • Clone the repository. If you are using eclipse, make sure that the project is imported from Git reopository.
  • Go through WEKA ecosystem.
  • This repository is used for classifying feature vectors on-the fly .
  • Configure build path appropriately.
